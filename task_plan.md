# 另一个我 (Another Me) - UI/UX 视觉与样式重构计划

## 项目背景
当前 UI 处于早期 MVP 阶段，缺乏“人生推演引擎”应有的沉浸感。
本计划旨在将其升级为“平行宇宙 / 赛博推演”风格的深色模式，并引入现代化的 CSS 架构（Tailwind CSS + shadcn/ui）。

## 核心视觉目标
1. **沉浸感深色模式**：太空灰 (`#0F172A`) 或 深渊黑 (`#09090b`) 背景。
2. **玻璃拟物化 (Glassmorphism)**：使用半透明背景和毛玻璃模糊效果 (`backdrop-filter: blur`)。
3. **科幻/发光点缀**：荧光青代表主线，紫罗兰代表变异，琥珀色代表转折。
4. **精细动画与交互**：组件悬浮反馈、D3 连线能量流转、打字机打字效果等。

## 阶段规划

### 阶段 1：基础架构重构 (CSS Architecture)
- [x] 移除现有 `App.css` 中的硬编码样式干扰。
- [x] 安装并配置 Tailwind CSS (Vite 环境)。
- [x] 安装并配置 shadcn/ui 基础组件库。
- [x] 配置全局深色模式 (Dark Mode) 及 CSS 颜色变量（适配赛博/推演引擎色系）。

### 阶段 2：全局布局与公共组件升级 (Layout & Common)
- [x] **AppShell & Sidebar**: 应用玻璃拟物化背景，移除生硬边框，通过色彩层级区分区域。
- [x] **Header**: 优化导航栏，引入毛玻璃吸顶效果。
- [ ] **表单与按钮 (Buttons & Forms)**：替换为 shadcn/ui 组件，添加发光阴影与微交互动效 (`hover: translateY`)。

### 阶段 3：核心推演可视化升级 (Core Visualizations)
- [ ] **DecisionTree (D3.js)**：
  - 连线：应用渐变色和虚线/能量流转动画。
  - 节点：选中时间线增加光晕 (Glow)，放弃的时间线降低透明度加虚线边框。
- [ ] **LifeChart (Recharts)**：
  - 将折线图 (LineChart) 改造为带有渐变填充的区域图 (AreaChart)。
  - 定制化具有科幻质感的暗黑 Tooltip。

### 阶段 4：功能组件与页面精修 (Pages & Features)
- [ ] **FutureLetter**: 引入打字机特效或代码控制台渐现动画，使用等宽字体 (Monospace)。
- [ ] **SimulatePage**: 优化决策输入框与戏剧性滑块的质感（金属质感或霓虹灯轨）。
- [ ] **响应式与排版优化**：调整最大内容宽度 (`max-width: 960px`)，优化留白呼吸感，数字使用等宽字体对齐。

## 当前重点提示
- 所有组件都应遵循无界感设计，减少实体边框，利用背景色差异和阴影区分层级。
